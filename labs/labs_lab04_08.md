## Create a Simple Java Triangle Program – 10 Points

**Points**: Complete this entire section for 10 points.

Create a Java class SimpleTriangle that has the following method

```java
public static void drawTriangle(int size) 
```

that draws the following triangle if called with an actual argument of 5.

```java
<>
<><>
<><><>
<><><><>
<><><><><>
```

The size of the triangle drawn is dependent on the value of parameter ```size```.  Work through several specific examples to discover the algorithm.  The algorithm I created is two for loops, one nested inside of the other.

You should call your ```drawTriangle``` method from a ```main``` method.
