## Move A Line Repeat

**Points**: Complete this entire section for 10 points.

Use the ```CGMoveALine.java``` file to accomplish the following.
* First execute the existing ```CGMoveALine.java```.
* Change the orientation of the line that is moved.  For example, draw a horizontal line.
* Study the code in ```CGMoveALine.java``` so that you understand it.
