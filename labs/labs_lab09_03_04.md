## Sort an Arbitrary Number of Strings

**Points**: Complete this entire section for 10 points.

Canvas Files->Labs->Searching->Sort.java contains the sort methods we developed in class, which sort int[].  Update the sort methods in Sort.java to sort ```ArrayList<String>```.  

Create a Java program that iteratively interacts with a user.  On each iteration, the user will enter a first name and Enter or just Enter.  Use a ```ArrayList<String>``` to store the first names.  When the user depresses Enter without a name, your program will sort the ```ArrayList<String>``` alphabetically and print the names alphabetically.  You should sort this ```ArrayList<String>``` with each of the sort methods in your updated ```Sort.java``` file.  The following is an example run of your program.

```java
Welcome to the Name Sorter.
Enter name: Gusty
Enter name: Zachary
Enter name: Emily
Enter name: Coletta
Enter name:
Coletta
Emily
Gusty
Zachary 
```
