## First Java Programming Lab

The First Java Programming Lab is your first chance to solve problems with Java.  The problems in this lab are not too difficult

##  Submission Requirements

Follow guidance in [Sample Lab Submission](/gustycooper.github.io/labs_lab00_00) for submitting your lab points.

## Install Java and BlueJ

**Points**: Install Java and BlueJ for 10 points.

1. Download and Install the Java SDK - see [Java Downloads](/gustycooper.github.io/mydoc_0_downloads).

2. Download and Install the BlueJ IDE - see [Java Downloads](/gustycooper.github.io/mydoc_0_downloads).

## Questions

**Points**: You get 1 point for each question that you write or type an answer, up to a maximum of 20 points.

1. What is a compiler?

2. What is an interpreter?

3. Is Java compiled or interpreted or both?  Explain your answer.

4. What is a CPU and what role does it serve in a computer?

5. What is RAM?

6. What is Secondary Storage?

7. What is the difference between a high-level language and a low-level (or machine language)?

8. Is Java high-level or low-level language?

9. True/False: Java is an expensive language?

10. Explain the phrase, “Java is platform-independent.”

11. Why do we normally use base 10?

12. Why does a computer use binary?

13. How many values does a byte hold?

14. How many values does a short hold?

15. How many values does an int hold?

16. True/False: A double and a long can hold the same number of values.

17. True/False: Memory is a sequence of bytes.

18. What is the binary equivalent of 10 decimal?

19. What is the decimal equivalent of 0b1010 binary?

20. True/False: In Java, a variable has a name, a value, a type, and associated memory locations.

21. What are the steps of a software development process?

22. What three steps of software development will this class focus on?

23. Define an algorithm.  Is a computer program an algorithm?  Can you have an algorithm that is not a computer program?  Is the following an algorithm?

    a. Roll a six sided die

    b. If the number rolled is between 1 and 6 repeat step a.

24. Define a data type.  Compare a Java byte and a Java short using your definition of a data type.  Explain a Java boolean type in terms of your definition.

25. Explain syntax and semantics of a programming language.  Give an example of a syntax error and a semantic error.
