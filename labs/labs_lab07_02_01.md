## Hello World GUI Repeat

**Points**: Complete this entire section for 5 points.

Use the ```HelloWorldGUI1.java``` and ```HelloWorldGUI2.java``` files to accomplish the following.

* First execute the ```HelloWorldGUI1.java``` and ```HelloWorldGUI2.java```.  
* Change the string in both of them so that it is ```“Hello UMW!”```.
* For ```HelloWorldGUI2.java```, you should change the name of the two ```private static``` classes.
  * ```HelloWorldDisplay``` – change to ```UMWWorldDisplay```
  * ```ButtonHandler``` – change to ```UMWButtonHandler```

Changing these will force you to study the code to see how these classes are used in the ```main()``` method.
