## EndorSmints

**Points**: Complete this entire section for 20 points.

The goal for this section is to place a GUI on your Endor Smints project.  If you followed the design presented in the Endor Smints project, creating a GUI is not too difficult.  You have two choices.  

1. Use the ```EndorGui.java``` class provided on Canvas.  You simply pair this class with your ```EndorSmints``` class, and you should have a GUI running rather quickly.  Study the ```EndorGui.java``` code so that you understand the design, how the design maps to the GUI, and how the design incorporates the EndorSmints.  For example, you should study all of the components and how they are placed in ```JPanels```, which are placed in the ```JFrame```.  The components are ```JLabel```, ```JButton```, ```JComboBox```, ```JCheckBox```.  You should also understand how the Action Listener is added to the components, how it is called, and how it determines what component has been selected
2. Implement your own GUI.  You can use the Netbeans GUI builder to drag and drop a GUI of your choice.  If you do this, most of the remaining steps will not apply.
