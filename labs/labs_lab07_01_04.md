## Simple Animation Starter

**Points**: Complete this entire section for 10 points.

Our textbook provides the ```class SimpleAnimationStarter``` in the file ```SimpleAnimationStarter.java```.  ```SimpleAnimationStarter``` is similar to ```GrowingCircleAnimation```; however, the skeleton code prints the frame number as it counts upwards.  Use ```SimpleAnimationStarter``` to construct your own simple animation.   The ```SimpleAnimationStarter.java``` on Canvas is the one demonstrated in class. You should think of some other animation.  Perhaps a filled circle that orbits some object. 

Using the ```SimpleAnimationStarter``` as the basis for Project 5 can be a good idea.  The following are examples created by former students.  Both projects are in ```Google Drive > 00UMW > 1Java > 0Class > SimpleGraphics```

* ```OrbitingPlanets.java``` - created by John Pruchnic
* ```ProjectGUI.java``` and ```CelestialObject.java``` - created by Billy McCord.  Billy began with John's code
