## Questions

**Points**: Answer these questions for 10 points. You have to handwrite or type the answers to get the points. For each question that you do not answer, subtract one point.


1. Describe checked and unchecked exceptions.

2. Give an example of a checked exception.

3. Give an example of an unchecked exception.

4. True/False: You must throw or catch an unchecked exception.

5. My code has a bug that generates a StringIndexOutOfBoundsException.  Is this a checked or unchecked exception?

6. My code attempted to open a file that was not available.  Is this a checked or unchecked exception?

7. True/False: Java is the only language that supports file I/O.

8. True/False: MS-Word is the only program that can read a file that has a .docx extenston.

9. True/False: I can use a Java ```Scanner``` to read information from a file.

10. True/False: Java's file I/O is easy to understand.
