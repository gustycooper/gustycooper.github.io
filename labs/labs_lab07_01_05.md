## Java Fonts

**Points**: Complete this entire section for 5  points.

Modify the ```RandomStringWithMain.java``` file.  The goal of this section is for you to manipulate Java ```Fonts```.  You should change the program so that it has 7 random fonts instead of 5.  You should also change the program so that it places 32 versions of the string in the window.
