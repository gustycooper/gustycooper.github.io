Reading materials

Operating Systems Textbooks

  • xv6: A simple, Unix-like teaching operating system by R. Cox, F. Kaashoek, and R. Morris.
    • Our primary textbook. Used for hands-on learning about operating systems.
  • OS Three Easy Pieces by Remzi H. Arpaci-Dusseau and Andrea C. Arpaci-Dusseau.
    • Our supplemental textbook. Used for various operating system concepts such as virtual memory, schedulers, and threads.
  • The Little Book of Semaphores by Allen B. Downey.
    • Used to understand semaphores in the barrier problem

UNIX

  • Youtube Unix intro
  • The UNIX Time-Sharing System, Dennis M. Ritchie and Ken L.Thompson, Bell System Technical Journal 57, number 6, part 2 (July-August 1978) pages 1905-1930.
  • The Evolution of the Unix Time-sharing System, Dennis M. Ritchie, 1979.

C Programming Language

  • The C Programming Language (second edition) Brian W. Kernighan and Dennis M. Ritchie. Prentice Hall, Inc., 1988. ISBN 0-13-110362-8, 1998.

RISC-V Emulation

  • QEMU - A fast and popular RISC-V platform and CPU emulator.
    • User manual
  • SiFive FU540-C000 Manual
  • Virtual I/O Device (VIRTIO)

RISC-V

  • The RISC-V Reader: An open architecture atlas by D. Patterson and A. Waterman, Strawberry Canyon, 2017.
  • RISC-V unprivileged instructions
  • RISC-V privileged instructions
  • Calling conventions
