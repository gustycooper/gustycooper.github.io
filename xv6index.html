CPSC 405 Home Page

  • The CPSC 405 Home Page is rather bland.

  • Fall 2024: CPSC 405 will be taught in Farmer-025, Tuesdays and Thursdays from 8:30am to 10:20am, starting on Tuesday Aug 26. CPSC 405 provides an introduction to operating systems. CPSC 405 is required for a CPSC major and is an elective for anyone interested in operating systems, which includes a CYBR major.

CPSC 405 Xv6 GitHub Repo

The starting source code for our CPSC 405 Xv6 labs is located at a GitHub repository. The repo has a branch for each lab. For example the util branch contains code for the Lab Utilities.
