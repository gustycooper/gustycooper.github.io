## Required Downloads

CPSC 220 uses Java 8 and two Interactive Development Environments: BlueJ and Netbeans.  We begin with BlueJ, which is a simple, easy-to-use IDE.  BlueJ allows students to see the relationship between classes.  BlueJ also forces students to understand the fact that Java must first be compiled into byte code prior to executing.  BlueJ can be a bit of a nuisance as you have to discover compile errors one at at time.  BlueJ also has the Codepad, which allows students to type Java code in a terminal.  This feature is great for early exploring of Java.  I still use the Blue Codepad whenever I have a question as to exactly what Java does.  We will use BlueJ for approximately the first 5 weeks of the class.  After five weeks, we shift to the Netbeans IDE, which is a full-flavored IDE used by many professionals.  Netbeans has more features than I understand.  Netbeans shows you compile errors as you type.  Netbeans also suggests code for you, which sometimes is not the code you desire.

## Download and Install Java SDK
Follow the instructions the following link to install the Java SDK. You want to download/install Jave Standard Edition, or Java SE.

[Java SDK](http://www.oracle.com/technetwork/java/javase/downloads/index.html)

You have to accept the Oracle License Agreement.  You will select one of the following to download and install.
Mac OS X x64	227.14 MB  	jdk-8u65-macosx-x64.dmg
Windows x64	186.57 MB  	jdk-8u65-windows-x64.exe – 64 bit

I doubt that anyone has a 32-bit version of Windows, but I think it is still supported.  If you have a 32-bit Windows, you download the following.
Windows x86	181.24 MB  	jdk-8u65-windows-i586.exe – 32 bit

## Download and Install the BlueJ IDE.
Follow the instructions at the following link to install BlueJ.  You will have to select the correct OS, which Windows, Mac OS X, or some flaovor of Linux.

[BlueJ](http://www.bluej.org)

You will have to select from Windows and Mac.  This web-site implies it will download both BlueJ and Java 8 SDK; however, I first downloaded Java (Step 1) followed by BlueJ.  For my Mac, BlueJ is simply a file in a folder that I double click to run.

## Download and Install the Netbeans IDE

You want to select Netbeans for Java SE.  You should select the latest version.  You will have to select the OS (Mac OS or Windows) using the platform pull-down menu.

[Netbeans](https://netbeans.org/downloads/)
