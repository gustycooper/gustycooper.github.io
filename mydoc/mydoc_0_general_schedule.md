## CPSC 220 General Schedule

The Textbook material contains everything we study.  Each expandable selection in the Textbook Sidebar is a module.  We cover the modules in order from top-to-bottom.  Within a module, you read each section from top-to-bottom.  Canvas contains the detailed schedule and due dates of readings and programming assignments. A rough schedule of the Textbook modules is provided in the following discussion.

The following are the core programming modules, during which you learn the essential problem solving and programming techniques for this course.  You will be tested on all of the programming techniques covered in these modules.

* Week 1
  * WebBook - [Intro & 1st Programs](/gustycooper.github.io/mydoc_1_overview)
  * Lab - [Intro and 1st Programs](/gustycooper.github.io/labs_lab01_00)

* Week 2
  * WebBook - [Expressions](/gustycooper.github.io/mydoc_2_expressions)
  * WebBook - [Methods](/gustycooper.github.io/mydoc_1a_methods)
  * Lab - [Exp, Methods](/gustycooper.github.io/labs_lab02_00)
  * [Project 1](/gustycooper.github.io/projects_methods) assigned

* Week 3
  * WebBook - [Simple Objects](/gustycooper.github.io/mydoc_3_simple_objects)
  * Lab - [Simple Objects](/gustycooper.github.io/labs_lab03_00)

* Week 4

The following module covers graphics, which are fun, but somewhat complex in Java.  Tests will ask general questions about graphics.  You will not have to program graphics on tests.

* Weeks 10, 11
  * WebBook - [Graphics](/gustycooper.github.io/mydoc_7_graphics)
  * Lab - [Graphics](/gustycooper.github.io/labs_lab07_00)
  * Project 4 due

The following module covers Big O, recursion, searching, and sorting.  You will have to solve recursive programming problems on tests.  You will have to program linear searching on tests.  Tests will ask general questions on Big O, searching and sorting.  You will not have to program binary searching and the various sorting algorithms we study.

* Weeks 12, 13
