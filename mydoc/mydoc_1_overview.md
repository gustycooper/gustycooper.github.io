## Introduction Overview

The Introduction section is primarily concerned with concepts and not programming.  The Introduction section defines the basic components of computers science.  After completing this section, you should understand the following concepts.

* computers
* input
* output
* CPU
* RAM
* secondary memory
* problems
* problem solving
* algorithms
* programs
* primitive types
* compiling
* interpreting
* assembly
* byte code

The Introduction Section does include some elementary programs, including a few programming problem solving patterns.

* Main Program Pattern
* Input Pattern
* Output Pattern
